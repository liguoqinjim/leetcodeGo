### [Two Sum](https://leetcode.com/problems/two-sum/#/description)
就是给你一个int数组和一个target，要找出数组里面哪两个数相加是等于target的。

|方法|简介|说明|
|---|---|---|
|twoSum||时间复杂度是O(n^2)|
|twoSum2||时间复杂度是O(n)|

#### 总结
我自己写的这个时间复杂度是O(n^2)，参考的算法的复杂度是O(n)。
参考算法用的是相减来查看有没有匹配的值
